# Mar Cerámicas 🌊🏺

Sitio web oficial de "Mar Cerámicas", una tienda de piezas de cerámica artesanal inspiradas en el mar.

---

## Contenido del Proyecto

- `index.html` → Página principal (presentación, galería, contacto)
- `tienda.html` → Página de tienda online (productos, carrito de compras)
- `estilos.css` → Estilos del sitio web (colores, fuentes, diseño)
- `carrito.js` → Funciones del carrito de compras (agregar, eliminar, finalizar compra)
- `imagenes/` → Carpeta donde van las imágenes de productos y galería.

---

## Instrucciones para correrlo

1. Crea una carpeta llamada `mar-ceramicas` en tu computadora.
2. Dentro de esa carpeta, guarda todos los archivos:
    - `index.html`
    - `tienda.html`
    - `estilos.css`
    - `carrito.js`
    - `README.txt`
3. Crea una subcarpeta llamada `imagenes` donde colocarás las fotos de productos y galería (pueden ser fotos tuyas o las que quieras usar).
4. Abre la carpeta en **Visual Studio Code**.
5. Haz clic en `index.html` y con botón derecho elige **"Abrir con Live Server"** (si tenés instalada la extensión).
   - Si no tenés Live Server, podés simplemente **abrir el archivo con tu navegador** (Chrome, Firefox, Edge).
6. ¡Listo! Verás tu web funcionando.

---

## Extras sugeridos
- Agregar nuevas imágenes en la galería y tienda.
- Personalizar más colores y fuentes en `estilos.css`.
- Instalar un sistema de pagos como Mercado Pago para ventas reales (requiere conexión a un backend).
